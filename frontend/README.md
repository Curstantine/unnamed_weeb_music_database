# Frontend

Contains the source code for the website.

# Prerequisites

- `node` v18.30.0<=

- `pnpm` v7.2.1<=

# Initialization

```bash
pnpm install
pnpm run dev
```
Preferred editor is vscode; and make sure to enable the [Take Over Mode](https://github.com/johnsoncodehk/volar/discussions/471) after installing the volar extension!
